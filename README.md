# art-kingshot

## Local Development

1. Ensure Node.js 20.19+ or 22.12+ is installed
2. Install client dependencies:
   ```bash
   cd client
   npm install
   cd ..
   ```
3. Install server dependencies:
   ```bash
   cd server
   npm install
   cd ..
   ```
4. Run the client:
   ```bash
   npm run dev:client
   ```
5. Run the server (in a separate terminal):
   ```bash
   npm run dev:server
   ```
